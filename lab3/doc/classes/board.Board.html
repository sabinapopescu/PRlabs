    Class Board

    Mutable ADT representing a Memory Scramble game board.

    A board is a rows×cols grid where each cell either:

    • holds a card with a picture (string)
    • is empty (null)

    Each card can be face-up or face-down. Face-up cards are controlled by the player who flipped them. Empty cells are always face-down with no controller.
